.. _install:

Installation
============
SKLL can be installed via ``pip``::

    pip install skll

or via ``conda``::

    conda install -c conda-forge -c desilinguist python=3.6 skll

It can also be downloaded directly from
`GitHub <https://github.com/EducationalTestingService/skll>`_.
